# JavaScript.Fun ![](https://github.com/im6/javascript-fun/workflows/build/badge.svg)

Code generator for [javascript.fun](https://www.javascript.fun/)

## Highlight

- react.js
- webpack
- express.js
- mySQL
- async
- [Pure.css](https://purecss.io/)

## Quick Start

create `.env` file for DB connections.

```sh
yarn install
yarn collect
yarn build
yarn daily
```
