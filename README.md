# JavaScript.Fun ![cicd](https://github.com/im6/javascript-fun/actions/workflows/ci.yml/badge.svg)

Source code to generate [www.javascript.fun](https://www.javascript.fun/)

## Quick Start

### Run app in local

```sh
yarn install
yarn dev
```

### Generate viewmodel

```sh
yarn build-misc
yarn collect
```

### Push to prod

```sh
yarn build
yarn render
yarn cp
```
